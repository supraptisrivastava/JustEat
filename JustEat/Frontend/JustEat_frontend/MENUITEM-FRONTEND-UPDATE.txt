🎯 HOW IT WORKS NOW:

1. User clicks "Add Menu Item"
2. Fills out form fields:
   - Item Name
   - Price
   - Description
   - Cuisine Type
   - Dietary Restriction
   - Mark as Special (checkbox)
   - **SELECT IMAGE FILE** ← New!
3. Clicks "Add to Menu"
4. Frontend creates FormData:
   - name
   - price
   - description
   - cuisineType
   - dietaryRestriction
   - isSpecial
   - image (FILE)
5. Sends to backend with multipart/form-data
6. Backend uploads to Cloudinary
7. Returns menu item with Cloudinary URL
8. Image displayed in menu list!
