🐛 ERROR YOU SAW:
   "Something went wrong: Invalid transformation parameter - {fetch"

🔍 ROOT CAUSE:
   The Cloudinary upload was using a nested "transformation" parameter
   with "fetch_format" which is not supported in the basic upload API.

✅ FIX APPLIED:
   Simplified the upload parameters to use direct options:

   BEFORE (incorrect):
   ─────────────────
   ObjectUtils.asMap(
       "folder", folder,
       "resource_type", "image",
       "transformation", ObjectUtils.asMap(
           "quality", "auto",
           "fetch_format", "auto"  ← This caused the error
       )
   )

   AFTER (correct):
   ────────────────
   ObjectUtils.asMap(
       "folder", folder,
       "resource_type", "image",
       "quality", "auto"  ← Direct parameter, no nesting
   )
